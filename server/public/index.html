Skills API Role in Trusted Career Profile (TCP)

The Skills API is the service layer that powers skills data for portable career credentials like TCP and CLR. It doesn't replace TCP - it feeds it.

üìä

Skills API

Service layer that manages, validates, and queries skills assertions across systems

‚Üí

Feeds Into

Provides structured skills data with evidence and proficiency levels

üé´

TCP / CLR

Portable, verifiable career credential containing skills, education, employment

Real-World Flow:

1 Person completes assessment, course, or gains work experience
2 Skills API creates verified skill assertions with evidence sources
3 Skills data is packaged into TCP or CLR as a portable credential
4 Person can share TCP/CLR with employers, systems validate using Skills API

Integration with JEDx API

The Skills API is designed to complement the HR Open JEDx (Job Exchange Data) format, creating a powerful ecosystem for skills-based hiring and workforce transformation.

‚ö° How They Work Together

JEDx Provides:
  • ‚Ä¢ Job postings structure
  • ‚Ä¢ Worker profile format
  • ‚Ä¢ Organization data
  • ‚Ä¢ Employment records
Skills API Adds:
  • ‚Ä¢ Detailed skill requirements
  • ‚Ä¢ Proficiency level measurements
  • ‚Ä¢ Evidence-based assertions
  • ‚Ä¢ Skills analytics & matching

Real-World Workflow:

1 Company posts job using JEDx format with position details
2 Skills API defines required skills with precise proficiency levels
3 Candidate's skills are assessed and mapped to the same proficiency scales
4 Analytics engine performs gap analysis and candidate matching
